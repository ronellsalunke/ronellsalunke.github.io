# Computers

### Work Laptop - Macbook Air (2020)

- CPU: Apple M1 (4P+4E)
- GPU: Apple 7 core
- RAM: 8gb 💀
- Storage: 256gb
- OS: MacOS 26

> While I really like the MBA M1 for a wide variety of reasons ranging from size and construction to the very satisfying keyboard, it is poorly suited to app development and MacOS tends to swap aggressively which is detrimental to long term usage

### Personal Laptop - Asus Tuf A15 (2020)

- CPU: Ryzen 5 4600H (6c 12t)
- GPU: Nvidia Geforce GTX 1650
- RAM: 16gb DDR4 3200
- Storage: 512gb NVME + 240gb SATA SSD
- OS: Windows 11 (planning yet another Linux install soon™️)

> The Tuf has essentially become by desktop, using it stationary and plugged to an external monitor. I do plan to eventually replace it but the hardware has been aging surpisingly well for the type of work I do
